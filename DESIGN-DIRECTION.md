# DESIGN-DIRECTION — ЛУМА (skin & body atelier)

Направление: **cinematic dark-stage** (вдохновлено референсом
`styles.refero.design/style/1bd3b2ba-...`, design system «Sequel») + один тёплый skincare-акцент.
Ощущение: премиально, спокойно, тактильно, медитативно. «Дорого через сдержанность».

## Исследование (конкуренты и принципы)

Разобраны премиальные skin-clinic / med-spa / wellness-сайты (Skinney MedSpa, Laser Clinique и
обзоры 2026: colorlib, sitebuilderreport, flamingoagency). Извлечённые принципы, которые применены:

1. **Тёмная «галерейная» сцена** вместо светлого «инстаграм-салона» — фото подсвечено как в кино.
2. **Дисплейный serif + геометрик-sans** — редакционная типографика = культурный авторитет.
3. **Full-bleed editorial image-cards** с оверлеем текста, бинарный радиус (12px карточки / 9999px pill).
4. **Огромный воздух** (section gap ~120px, max-width 1280) — ощущение дороговизны через пустоту.
5. **Почти ноль цвета** + единственный тёплый акцент → контроль и премиальность.
6. **Динамичный hero** (видео/анимация/parallax), а не статичный баннер.
7. **Доверие через прозрачность**: диагностика, честный план, открытый прайс, один мастер.

Чего избегали: кремовый фон + терракот + Playfair («AI-вид»), эмодзи, длинное тире, layout
«hero + 3 карточки», стоковые «улыбки в халатах», агрессивные продажи и обещания «минус 10 лет».

## Типографика

- **Display:** Cormorant (serif, Cyr ✓, OFL) — H1 `clamp(3.25rem, 11vw, 8.5rem)`, tracking -0.03em, lh ~1.0.
- **Body/UI:** Onest (sans, Cyr ✓, OFL) — 16–18px, вес 300–400.
- **Meta/индексы:** JetBrains Mono — 11–12px, uppercase, letter-spacing 0.08–0.18em.
- Self-host woff2 (cyrillic + latin сабсеты), `font-display: swap`, preload ключевых начертаний.

## Hero — 3 концепции

**Реализована №1 «Lit Subject».** Full-bleed тёмное портрет-фото с медленным Ken-Burns, тёплый
радиальный «свет», cursor-follow подсветка, заголовок «Кожа любит тишину» появляется по словам
(mask/stagger), плавающие pill-карточки направлений с parallax, при скролле hero затемняется и
масштабируется (хэндофф в «Философию»). Mobile: статичнее, без курсор-параллакса; reduced-motion: fade.

- **№2 «Floating Atelier».** Сетка из 3–4 медленно дрейфующих editorial-карточек (лицо/тело/массаж/
  текстура), заголовок слева, scroll-driven перестроение карточек в ленту направлений.
- **№3 «Texture Reveal».** Крупная макро-текстура (крем/вода/кожа) на фоне, маска-blur расходится от
  центра при загрузке, заголовок проявляется из текстуры; скролл переводит макро в портрет.

## Арт-дирекшн фото

Источник: **Pexels** (free commercial license, без атрибуции). 19 изображений в `public/img/`.
Стиль: тёмный, moody, один тёплый источник света, tight framing, десатурация. Портреты — модели
европейской/славянской внешности (по требованию клиента). Поисковые запросы, которые сработали:
`woman portrait dark moody beauty`, `blonde woman beauty face clean skin`, `cream texture macro`,
`massage therapy spa`, `luxury spa interior`, `facial treatment`.

Категории: hero-портрет, лицо, тело, массаж, проблемная кожа, антивозраст, серум/текстуры, интерьер,
специалисты. Избегать: яркие цветные фоны (синий/неон) — выбиваются из ахроматичной сцены.

## Анимации

- UI-переходы < 300ms, ease-out, только transform/opacity.
- Page-load hero: один оркестрированный stagger (слова 0.12s, элементы 0.12s).
- Reveal секций: IntersectionObserver, translateY(24px)+opacity, 0.8s ease-out, stagger через `--reveal-delay`.
- `prefers-reduced-motion` обязателен — гасит Lenis, Ken-Burns, float, reveal.
